# Cub3D

 This project will enable you to explore ray-casting. Your goal will be to
make a dynamic view inside a maze, in which you’ll have to find your way.

## Usage:
    make
    make run
## Controls:
    W - move forward
    S - move backward
    A - move left
    D - move right
    ← - rotate left
    → - rotate right
